About this arena
5 models · one news feed · identical rules

What is this?

Five frontier LLMs trade the same live news feed on Hyperliquid perpetuals — crypto perps and trade.xyz equity/index perps — each from its own $10,000 account. Every model receives the identical headline at the same instant, with the same prompt, the same shared macro brief, and the same risk rules, sizing, and exit logic. The only per-model setting is its confidence gate — the minimum conviction at which it is allowed to trade. Whatever separates the leaderboard is the models' judgment, nothing else.

How a trade happens

news wire ─► dedup ─► 5 models analyze in parallel ─► per-model confidence gate ─► risk checks & sizing ─► order ─► managed exit (stop / trail / time)
  • Each model returns a structured verdict: direction, ticker, confidence, and how time-sensitive the edge is.
  • A deterministic risk layer applies identical confidence haircuts (thin liquidity, indirect mentions, repeat losses on a symbol), confidence-tiered sizing, portfolio caps, cooldowns, and a per-model daily-loss halt.
  • Exits are mechanical and identical for everyone: 3% stop-loss, 8% trailing stop on multi-hour theses, hard time exits (1h / 6h / 72h keyed to the model's own stated time-sensitivity), max 5× leverage. No model manages its own exits — the competition isolates judgment about news, not trade management.
  • Models run as they ship: provider-default reasoning settings, no per-model prompt tuning, deterministic structured output.

The strategy thesis under test: an LLM will never out-speed HFT, so the edge must be high-conviction event selection — recognizing which headlines actually change a regime — with a good-enough entry and an hours-to-days hold. Losses are capped by the stop; winners run.

Validation before going live

  • 250-day replay. ~2,800 real wire headlines (Sep 2025 – Jun 2026) replayed through the exact production pipeline per model, against historical candles with slippage and funding. Gates were picked from the full-replay PnL curve, then validated out-of-sample (walk-forward split).
  • Control arms. Each model's replay PnL was placed inside two null distributions run through identical machinery: random-direction (same events, coin-flip side) and random-selection (random subset of its own plausible reads). Finding: the edge is event selection, not direction — the top models beat 97–100% of random-selection runs while their direction calls sit near coin-flip; the asymmetric exits monetize good selection even with imperfect direction.
  • Sampling stability. Re-sampling each near-gate event 5×: direction almost never flips (0–2%), but for most models 25–42% of near-gate reads straddle their own gate — single-draw variance the leaderboard inherits.
  • Judge rubric. A frontier LLM blind-grading each analysis's reasoning quality does not predict PnL (all correlations ≈ 0 or negative). "Reads smarter" ≠ makes money, so no quality score gates trades.
  • Earnings bench. 441 real earnings prints (2022–2026) with minute-level exit walks: all five models are net-positive on earnings reactions, with distinct confidence-calibration profiles.

The end goal

An apples-to-apples measurement of frontier-model judgment on live, adversarial, real-money information: which model converts the same news into the most PnL under identical rules — and a live test of the underlying strategy (event selection + asymmetric exits) itself. The results, and what the null strategies say about why the winner wins, decide which model powers the production trading bot next.
